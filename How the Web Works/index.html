Question 1: Using curl to make a GET request

Open a command prompt and run the following command:

curl -X GET \ https://icanhazdadjoke.com/search \ -H 'Accept: application/json' \ -d 'term=pirate'

Question 2: Finding the IP address of icanhazdadjoke.com

Open a command prompt and run the following command:

dig +short icanhazdadjoke.com

For Windows:

nslookup icanhazdadjoke.com

Example output:

        Server: Unknown
        Address: 192.168.100.1

        Non-authoritative answer:
        Name: icanhazdadjoke.com
        Addresses: 104.21.66.15 172.67.198.173
